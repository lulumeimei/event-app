# Ticketapp

#### How to config env (staging / production)
```
// app_config.dart
// line 46 in main.dart
appConfigInterface = kReleaseMode ? AppConfig.prod() : AppConfig.dev();
```

#### How to run the project
```
flutter pub get
flutter run
```
